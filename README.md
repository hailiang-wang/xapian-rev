# xapian-rev

Build xapian releases for ubuntu x86_64.

## Sources
Download from https://xapian.org/download

## Install script

Launch docker.

```
./scripts/dev.sh
```

Run build in docker.

```
/workspace/scripts/build.sh
```

Get new release.

```
build/
```
